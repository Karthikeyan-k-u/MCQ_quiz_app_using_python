# MCQ_Quiz_app_using_python
This is a console-based MCQ quiz program in PYTHON that asks 20 multiple-choice questions. The user enters answers using capital letters. The program checks each answer, calculates the total score, and shows the correct answers for wrong questions. At the end, the user can choose to play the quiz again or exit.

**Description of the MCQ Quiz Application**
            This program is a console-based MCQ Quiz APP written in PYTHON using only decision statements and loops, without using functions, import statements, or the main function.
The quiz consists of 20 multiple-choice questions from general knowledge, science, mathematics, and basic programming concepts. The user is required to enter their answers using capital letters (A, B, C, or D).
At the start of each quiz attempt, the program initializes the score and resets an array to track incorrect answers. Each question is displayed one by one, and the user’s response is checked using if–else statements. If the answer is correct, the score is increased; otherwise, the corresponding question number is marked as wrong.
After all 20 questions are completed, the program displays:
The final score out of 20.A list of questions answered incorrectly, along with their correct answers.If the user answers all questions correctly, a congratulatory message is shown.Finally, the program asks the user whether they want to repeat the quiz. If the user enters 1, the quiz restarts; if 0, the program ends with a thank-you message.
